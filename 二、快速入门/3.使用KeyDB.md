# 使用 KeyDB

这个数据库是redis的一个高性能分支，核心函数集保持不变（请参阅此处的解释）。这使您能够像使用Redis一样使用KeyDB，
但具有性能提高和简化 设置/配置 选项的优势。

KeyDB仍然是Redis当前版本的最新版本，因此保持与标准Redis API和协议的兼容性。
前端工具、redis-cli API 和 模块将与KeyDB一起工作。KeyDB可以放在适当的位置，只有KeyDB需要配置的新特性。

这里展示的文档着眼于Redis或与Redis相关的工具中尚未概述的修改使用。

有关常用命令，请访问redis.io/commands
