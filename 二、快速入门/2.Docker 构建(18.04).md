# 适用于Ubuntu18.04 的Docker 构建

如果您正在Docker上运行，请参阅Docker部分。

# 生成最新的二进制文件（Ubuntu 18.04）

如果您想要最新的二进制文件，但不想自己运行构建，可以在docker容器中生成。请注意这是在Ubuntu 18.04中构建的。如果你想建立在一个不同的图像上，dockerfile会被发布在Docker部分。

创建一个要转储最新二进制文件的文件夹，然后使用更新的路径运行以下命令：

    $ docker run -it --rm -v /path-to-dump-binaries:/keydb_bin eqalpha/keydb-build-bin

您应该收到以下文件：keydb-benchmark、keydb-check-aof、keydb-check-rdb、keydb-cli、keydb-sentinel、keydb-server

# 开启Flash 支持

如果您希望通过构建（make MALLOC=memkind）启用flash支持，请使用以下命令：

    $ docker run -it --rm -v /path-to-dump-binaries:/keydb_bin eqalpha/keydb-build-bin:flash
